Spacing & Grid

Holy uses an 8px base unit. The grid system scales from a 10-column mobile layout to a 20-column desktop layout with a maximum container width of 1600px.

Border Radius

Holy uses 0px border radius throughout — all buttons, cards, inputs, and containers have sharp corners. This creates a bold, structured, editorial aesthetic consistent with the brand's energetic visual language.
